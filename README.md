Introduction:
----------
Scans /proc/id/maps file and adds two columns
- Size in bytes
- Size in human readable form (B, KB, MB, GB)
Then passes it to output

Usage:
----------
`python ProcMapsSizeDump.py < /proc/id/maps`
`dumpit() { grep $REGION /proc/$1/maps | sed -n 's/^\([0-9a-f]*\)-\([0-9a-f]*\) .*$/\1 \2/p' | while read start stop; do gdb --batch --pid $1 -ex "dump memory $1-$start-$stop.dump 0x$start 0x$stop"; done ;}; dumpit $PID`
